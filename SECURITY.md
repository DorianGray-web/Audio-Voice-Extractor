# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| 1.x | ✅ |

---

## Reporting a Vulnerability

Although this project processes audio files locally and does not interact with external services, security issues are taken seriously.

If you discover a vulnerability, please open a GitHub Security Advisory or contact the author before creating a public issue.

---

## Notes

This project:

- does not collect user data;
- does not connect to external servers;
- does not transmit audio recordings;
- processes all files locally on the user's computer.
